State preferences as one concrete rule per file, named by the words you'd search for.

**Why:** vague filenames ("note 3") never get retrieved; the filename and description are
the surface that search and recall match against.
**How to apply:** when the user corrects how you work, capture the rule plus the why here,
then link related notes with [[example-reference]].
